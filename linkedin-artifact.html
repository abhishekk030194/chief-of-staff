The Problem

Life is chaotic. We needed a shared brain.

Between managing work, household tasks, shopping, and keeping each other in the loop — things kept falling through the cracks. Tasks lived in one app, the shopping list in another, market updates in a third browser tab, ideas scattered in notes. I wanted a single intelligent assistant that both of us could talk to naturally and that actually took action — not another app to maintain.

How We Built It

Started with a conversation, not a spec

No wireframes, no technical planning. Just described what I wanted to Claude Code: "I want an AI assistant for me and my wife that handles tasks, scheduling, shopping, and household management."

Picked the right interface: Telegram

Chose Telegram because it works on both our iPhones, has an excellent bot API, supports voice messages, and feels like a natural chat — not another productivity app.

Claude Code wrote the entire bot

Using Claude Code, the complete bot was built — Telegram integration, Claude AI calls, task management, conversation memory, group chat, and Google Calendar. No manual coding needed.

Added voice, Notion, and smart digests

Iterated quickly — added faster-whisper for voice transcription, Google Calendar for iPhone reminders, Notion as a live database, and a daily 7 AM digest with real-time stock market indices.

Added security, observability, and auto-recovery

Prompt injection firewall, secret scanner, real-time Telegram security alerts, Notion security log. Daily cost reports, eval reports, /status command. Retry logic, graceful shutdown with Notion sync.

Long-term memory and image analysis

Mira now remembers facts about the family across conversations — stored in a Notion Memory database, auto-extracted after each chat. Added Claude Vision so you can send any photo (receipt, document, shopping list) and Mira reads and acts on it.

Status notifications and 24/7 cloud deployment

Mira now sends Telegram alerts when she goes online (🟢), shuts down (🔴), or wakes from sleep (😴). Deployed to Railway cloud — runs 24/7 independent of the Mac, with voice support via faster-whisper in a Docker container.

Cross-user notifications — Mira as the messenger

When one person gives Mira an instruction with an action for the other (e.g. "remind Abhishek to buy chocolates"), Mira instantly sends a direct Telegram message to that person. No more relying on "I told Mira to tell you."

What Mira Can Do
📋

Shared Task Management

Both of us can add, view, and complete tasks. Tracks who added what and syncs to Notion automatically.

📅

Google Calendar Events

Say "remind me at 4:30am Sunday for my marathon" — it creates the event with alerts straight to your iPhone.

🎙️

Voice Messages

Send a voice note in Telegram. Whisper transcribes it, Claude understands it, and action is taken automatically.

🛒

Smart Shopping List

Say "add milk, eggs and bread" — items appear on the shared list and sync to Notion instantly.

💡

Ideas Capture

Save blog ideas, AI product concepts, and more. Mira auto-tags each idea by type and stores it in Notion.

📈

Live Market Digest

Every morning at 7 AM IST, get real-time prices and % change for S&P 500, NASDAQ, Nifty 50, Sensex, Gold, and Silver.

🗃️

Notion Sync

All tasks, shopping items, and ideas auto-sync to Notion databases in real time — accessible anywhere.

🔍

MacBook File Search

Ask Mira to find any file on your MacBook. It searches and returns the path directly in chat.

🧠

Long-term Memory

Remembers facts about the family across all conversations — allergies, goals, important dates. Auto-extracted after every chat and stored in a Notion Memory database.

🖼️

Image & Photo Analysis

Send any photo — a grocery receipt, a bill, a handwritten list, a document. Claude reads it and takes the right action automatically.

🔐

Security Firewall

Prompt injection detection, secret leak scanner, real-time Telegram security alerts, and a Notion security log for every access attempt.

📡

Observability & Reports

Daily cost reports (tokens, $), eval reports (actions taken, errors), live /status checks — all synced to Notion automatically.

⚙️

Auto-Recovery

Retries failed API calls with exponential backoff, syncs state to Notion on shutdown. Hosted on Railway for 24/7 uptime.

⌨️

Command Autocomplete

Type / in Telegram to see all 18 commands with descriptions instantly — no need to remember syntax.

👥

Family Group Chat

Works in a shared Telegram group. Doesn't interrupt every message — only responds when relevant.

🟢

Status Notifications

Mira sends a Telegram message when she comes online, shuts down, or wakes from Mac sleep — so you always know her state.

☁️

24/7 Cloud Hosting

Deployed on Railway — runs always, independent of any Mac. Docker container with ffmpeg included for voice. ~$0–1/month.

🔔

Cross-User Notifications

When Alekya says "remind Abhishek to buy chocolates", he gets an instant Telegram DM. Works both ways — frictionless coordination between partners.
