The Problem

Life is chaotic. We needed a shared brain.

Between managing work, household tasks, shopping, and keeping each other in the loop — things kept falling through the cracks. Tasks lived in one app, the shopping list in another, market updates in a third browser tab, ideas scattered in notes. I wanted a single intelligent assistant that both of us could talk to naturally and that actually took action — not another app to maintain.

How We Built It

Started with a conversation, not a spec

No wireframes, no technical planning. Just described what I wanted to Claude Code: "I want an AI assistant for me and my wife that handles tasks, scheduling, shopping, and household management."

Picked the right interface: Telegram

Chose Telegram because it works on both our iPhones, has an excellent bot API, supports voice messages, and feels like a natural chat — not another productivity app.

Claude Code wrote the entire bot

Using Claude Code, the complete bot was built — Telegram integration, Claude AI calls, task management, conversation memory, group chat, and Google Calendar. No manual coding needed.

Added voice, Notion, and smart digests

Iterated quickly — added OpenAI Whisper for voice, Google Calendar for iPhone reminders, Notion as a live database, and a daily 7 AM digest with real-time stock market indices.

Polished and open-sourced

Added security (only family members can use it), command autocomplete, IST timestamps, file search, and ideas capture. Code is open-source on GitHub.

What Mira Can Do
📋

Shared Task Management

Both of us can add, view, and complete tasks. Tracks who added what and syncs to Notion automatically.

📅

Google Calendar Events

Say "remind me at 4:30am Sunday for my marathon" — it creates the event with alerts straight to your iPhone.

🎙️

Voice Messages

Send a voice note in Telegram. Whisper transcribes it, Claude understands it, and action is taken automatically.

🛒

Smart Shopping List

Say "add milk, eggs and bread" — items appear on the shared list and sync to Notion instantly.

💡

Ideas Capture

Save blog ideas, AI product concepts, and more. Mira auto-tags each idea by type and stores it in Notion.

📈

Live Market Digest

Every morning at 7 AM IST, get real-time prices and % change for S&P 500, NASDAQ, Nifty 50, Sensex, Gold, and Silver.

🗃️

Notion Sync

All tasks, shopping items, and ideas auto-sync to Notion databases in real time — accessible anywhere.

🔍

MacBook File Search

Ask Mira to find any file on your MacBook. It searches and returns the path directly in chat.

🧠

Context Memory

Remembers recent conversations so follow-up questions feel natural. Knows who's talking and responds accordingly.

⌨️

Command Autocomplete

Type / in Telegram to see all 12 commands with descriptions instantly — no need to remember syntax.

🔐

Family-Only Security

Only authorised Telegram users can interact with Mira. Everyone else is silently ignored.

👥

Family Group Chat

Works in a shared Telegram group. Doesn't interrupt every message — only responds when relevant.
